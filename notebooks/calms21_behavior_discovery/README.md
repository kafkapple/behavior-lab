# CalMS21 Behavior Discovery: Map of Content (MoC)

This directory serves as the centralized experiment space for unsupervised behavior discovery pipelines on keypoint data. It unifies four discovery methods (SUBTLE, B-SOiD, keypoint-MoSeq, BehaveMAE) across multi-scale feature spaces.

---

## 🗺️ Notebook Roadmap (SSOT)

| Notebook | Focus | Primary Feature Space | Key Finding / Outcome |
|---|---|---|---|
| **[Notebook 1 — SUBTLE Baseline](01_subtle_baseline.ipynb)** | Baseline spontaneous repertoire discovery | Single-mouse pixel coords (12-dim) | Established baseline; highlighted the need for inter-subject features due to low initial GT alignment. |
| **[Notebook 2 — Dyadic Features & Comparison](02_dyadic_features_and_comparison.ipynb)** | Multi-method benchmark & visualization | 24-dim Ego-centric Dyadic features | Dyadic representations resolve social context. Compared SUBTLE, B-SOiD, and HMM Fallback; generated per-cluster bout GIFs. |
| **[Notebook 3 — Real keypoint-MoSeq Sticky HDP](03_real_kpmoseq_sticky_hdp.ipynb)** | Temporal persistence & state fragmentation | 6-joint Ego-aligned coords (12-dim) | Sticky HDP prior ($\kappa=10^6$) avoids state fragmentation, resulting in 6-12× longer median bout length compared to vanilla HMM. |
| **[Notebook 4 — BehaveMAE Hierarchical Discovery](04_behavemae_hierarchical_discovery.ipynb)** | Multi-scale representation & hierarchical clustering | 4D skeleton keypoints (MABe22) | Extracted Stage 1 (Pose), Stage 2 (Locomotion), and Stage 3 (Action) features, visualizing the hierarchy of behavioral semantics. |

